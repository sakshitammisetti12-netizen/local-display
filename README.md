# 🌡️ Temperature and Humidity Display System (Wokwi)

## 📌 Overview

This project demonstrates a temperature and humidity monitoring system using Arduino, simulated in Wokwi. The system reads environmental data from a sensor and displays real-time temperature and humidity values on an LCD screen.

## ⚙️ Components Used

* Arduino Uno
* DHT11 / DHT22 Sensor
* 16x2 LCD Display
* Resistors
* Breadboard
* Connecting Wires

## 🔌 Working Principle

The DHT sensor measures temperature and humidity from the environment and sends the data to the Arduino. The Arduino processes this data and displays the values on the LCD screen in real time.

## 💻 Code

The Arduino program uses the DHT library to read sensor data and the LiquidCrystal library to control the LCD display. The values are updated continuously.

## 🧪 Simulation

This project is created and tested using the Wokwi simulator, allowing real-time interaction and visualization of sensor data.

## 📸 Output

The LCD displays:

* Temperature (°C)
* Humidity (%)
